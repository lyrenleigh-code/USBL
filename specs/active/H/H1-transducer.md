# H1: 接收换能器

> 模块线：H（硬件）
> 依赖：[[yumin-2006-lr-usbl]]、[[usbl-hardware-spec]]、[[neub-816-test-report-260102]]
> 里程碑映射：HM1、HM2

## 目标
**12 kHz** 中心频率、≥4 kHz 宽带接收换能器单元，耐压 40 MPa。
对应供应商产品型号 **NeUB-816**（江苏水声技术有限公司）。

## 范围
- 包含：复合棒 PZT4 方案 + 匹配层 + 尾质量 + 灌封
- 包含：有限元仿真、单元电声测试
- 包含：备选方案 1-3 压电复合材料评估
- 不包含：阵列封装（H2）、发射换能器（H3）

## 输入接口
- 算法侧冻结的信号参数：**f0 = 12 kHz, BW = 4 kHz**（来自 A1；2026-04-19 从 10 kHz 更新）
- 链路预算反推灵敏度：≥ −200 dB re 1V/μPa（按实测值冻结）

## 输出接口
- 换能器样品 N = 10（供 H2 筛选）
- 电声测试报告（灵敏度/阻抗曲线/方向图）
- 幅相一致性原始数据（供 M2）

## 验收标准
- [x] 灵敏度 ≥ −200 dB re 1V/μPa（8-16 kHz 内变化 < 1 dB）— ✅ **实测达标**
- [ ] −3 dB 带宽 ≥ 4 kHz（覆盖 10-14 kHz 工作段）
- [ ] 耐压 40 MPa（1.25× 余量打压）— **⚠️ NeUB-816 测试报告未含耐压测试，需单独安排**
- [x] 尺寸约束：**阵列整体** Φ204 × 233 mm（非单元尺寸，见工程图 [[five-element-transducer-assembly-drawing-vA]]）— 实测达标
- [x] 批次一致性：5 通道幅度差 ≤ 1 dB（实测 ±0.5 dB）— ✅ 达标

## 供应商实测参数（NeUB-816 样机 1/2，测试日期 2026-01-24）

### 接收灵敏度（8-16 kHz，近乎平坦）

| f (kHz) | M (dB) | f (kHz) | M (dB) |
|---------|--------|---------|--------|
| 8.0  | -199.8 | 12.0 | **-199.3** |
| 10.0 | -199.2 | 14.0 | -199.5 |
| 16.0 | -199.9 |      |        |

**结论**：全频段内优于 spec 原目标 -190 dB **约 10 dB 裕量**；新 spec 按实测冻结为 -200 dB。

### 通道一致性（5 通道，12 kHz）

- 样机 1（260102-1）：幅度 ±0.5 dB，相位 ±3.6°
- 样机 2（260102-2）：幅度 ±0.5 dB，相位 ±3.1°

详见 [[neub-816-test-report-260102]] 完整通道校正表（供 M2 软件校正使用）。

### 指向性（12 kHz）

- 水平指向性：近似全向（水平面内响应差 < 3 dB）
- 垂直指向性：cardioid 型，主瓣在阵法线方向（0°），180° 方向有明显凹陷

**该垂直非对称性佐证笼式立体阵结构**（4 外围立柱 + 1 中央）。
