# M2: 电声一致性标定

> 模块线：M（测量与校准）
> 依赖：[[usbl-hardware-spec]]、[[dingjie-2020-compact-usbl]]、[[neub-816-test-report-260102]]
> 里程碑映射：HM3、HM4

## 目标
测量各阵元+通道的幅度/相位响应，生成校正表，使等效一致性达到 DOA 要求。
**路径决策（2026-04-19）**：走**软件校正**（不再要求硬件侧一致性提升）。

## 范围
- 包含：逐元发射响应测试、阵列接收一致性测试、通道级电子增益/相位测试
- 包含：频段内（**12 ± 2 kHz**）幅相校正表生成（2026-04-19 从 10 ± 2 kHz 更新）
- 包含：温度稳定性测试
- 包含：**消费供应商 NeUB-816 测试报告的原始校正数据**（2026-04-19 新增）
- 不包含：阵型几何（M1）、年度复校触发逻辑（M5）

## 输入接口
- H2 阵列整机
- H4 采集电子
- 标定信号源（宽带扫频）
- **[[neub-816-test-report-260102]] 供应商校正源数据**（2026-04-19 并入）

## 输出接口
- 幅相校正表 `array_cal.phase_amp[5][freq]`
- 校正后一致性测试报告
- 温度漂移曲线

## 验收标准

### 原始一致性（由供应商报告直接填入）
- [x] 校正前幅度一致性 ≤ 1 dB — ✅ 实测 ±0.5 dB
- [ ] 校正前相位一致性 ≤ 5° — ⚠️ **实测 ±3.6°（接近上限但未超）**，与原 spec 一致

### 校正后（软件路径）
- [ ] 校正后幅度残差 ≤ **0.3 dB**（保持原 spec）
- [ ] 校正后相位残差 ≤ **1°**（保持原 spec）
- [ ] DOA 算法使用校正表后 @SNR=10dB 精度 ≤ 0.3° RMS（与 A2 验收对齐）
- [ ] 温度 −5~+40°C 漂移 ≤ 2° 相位

### 路径说明

**原路径**（硬件一致性 ≤ 1°）——已放弃
- 要求供应商提升一致性成本高、周期长
- 硬件一致性天然受制于 PZT4 材料批次差异

**新路径**（软件校正表反补）——采纳
- 用供应商 12 kHz 单点实测数据作为基础校正表
- 扩展到 8-16 kHz 全频段（需自行扫频实验，或要供应商补测）
- 校正后由 DOA 残差验证（见 A2）

### 校正表构造（MATLAB 伪代码）

```matlab
% 从供应商单点数据生成初版校正表（仅 12 kHz 点）
function cal = build_cal_table_v0(unit_id)
    switch unit_id
        case 260102-1
            cal.channel_ids = {'1', '2', '4', '6', '8'};
            cal.freq_Hz = 12000;
            cal.amp_dB   = [0.0, 0.2, 0.1, 0.3, 0.5];    % 幅度差
            cal.phase_deg = [0.0, 3.4, 3.6, -2.4, -2.6];  % 相位差
        case 260102-2
            cal.channel_ids = {'3', '5', '7', '11', '14'};
            cal.freq_Hz = 12000;
            cal.amp_dB   = [0.0, 0.5, -0.3, 0.2, 0.4];
            cal.phase_deg = [0.0, 3.1, -2.9, -1.6, -1.4];
    end
    % 应用方式：RX 各通道信号 × 10^(-amp_dB/20) × exp(-j*phase_deg*π/180)
end
```

### 扩展到全频段的路径

- **路径 A**（自行扫频）：M2 标定实验用 8-16 kHz 扫频 + 台架 ULA 源 → 生成全频段校正表
- **路径 B**（要求供应商补测）：向江苏水声提交补充测试请求，测 {8, 10, 12, 14, 16} kHz 多点
- **路径 C**（插值近似）：12 kHz 单点外推（假设 PZT4 相位响应在邻近频段线性）— 仅用于 MVP
