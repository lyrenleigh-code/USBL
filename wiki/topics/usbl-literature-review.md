## 六层研究体系

```
第一层：声学基阵与信号处理（物理基础）
  ├─ 基阵构型：十字/圆阵/立体阵/非等距 [蘧振超, 何旭涛, 丁杰]
  ├─ 信号体制：窄带相位差 / 宽带时延差 / 组合策略 [喻敏]
  ├─ DOA估计：ML/基线分解/多阵型融合/GCC [丁杰, 蘧振超, 黄健]
  └─ 抗相位模糊：脉冲对/解缠绕/全局搜索ML [郑翠娥, 本项目]

第二层：声速修正与声线跟踪（环境适配）
  ├─ 有效声速：等效声速/BELLHOP迭代/PSO优化 [杨保国, 黄健]
  ├─ 声线跟踪：常梯度分层/自适应分层 [杨保国]
  └─ 声速剖面：温盐压公式/平滑/拓延 [杨保国]

第三层：系统误差校准（工程保障）
  ├─ 阵型误差：水池校准/有效声速+基线重构 [喻敏, 丁杰]
  ├─ 安装偏差：两步校准/矩阵分解解耦/M估计 [杨保国, 丁杰]
  └─ 校准航迹：对称设计/多距离验证 [杨保国, 喻敏]

第四层：定位解算与滤波（算法核心）
  ├─ 单次解算：相位差→方位→坐标 / 近远场模型 [基础, 郑翠娥]
  ├─ 跟踪滤波：EKF/UKF/R-T-S平滑/IMM [何旭涛, 黄健]
  └─ 异常值：Grubbs准则/η剔除/鲁棒最小二乘 [黄健, 蘧振超, 郭瑜]

第五层：多系统组合导航（系统集成）
  ├─ SINS/DVL/USBL融合：松紧组合/李群IEKF [郭瑜]
  ├─ LBL/USBL异构融合：距离加权/冗余声速反演 [黄健]
  └─ USBL失效处理：神经网络预测+信息重构 [郭瑜]

第六层：硬件实现与系统集成（工程落地）
  ├─ 换能器：复合棒PZT4/阵列集成/耐压4000m [喻敏]
  ├─ 应答器：CW/chirp/扩频/多目标CDMA [喻敏, 郑翠娥]
  └─ 集成化：GAPS模式(USBL+GPS+INS一体化) [丁杰, 喻敏]
```

## 待解决问题（文献未充分覆盖）

1. **iUSBL 自定位坐标变换**：现有文献绝大多数是正向 USBL，逆向变换链需重新推导
2. **五元圆阵特殊性**：文献多针对十字阵/球面阵，UCA 模态分解适用性待验证
3. **10kHz 低频长程**：多数商用设备 14kHz+，10kHz 海洋噪声更大
4. **d/λ=1.33 模糊处理**：ML 全局搜索可天然解模糊，需实测验证
5. **单快拍 ML 鲁棒性**：理论最优但对模型失配敏感
