# 长程 USBL 定位系统研制计划

> 基于 [[usbl-literature-review]] 文献调研制定

## 核心指标

| 指标 | 要求 | 对标设备 |
|------|------|---------|
| 作用距离 | ≥ 10 km | HiPAP 102 / Posidonia |
| 定位精度 | < 1%R | Posidonia (0.2%R) |
| 频率 | 10 kHz (λ=15cm) | HiPAP 102 (10-15kHz) |
| 阵型 | 五元 UCA, d=20cm | 自研 |
| 信号 | LFM, BW=4kHz, T=100ms | — |
| 模式 | 逆向 USBL 自定位 | — |

## 四阶段路线图

| 阶段 | 内容 | 关键交付 |
|------|------|---------|
| **Phase 1（当前）** | 算法原型完善 | 全链路 MATLAB 仿真 |
| **Phase 2** | 系统仿真验证 | Monte Carlo 精度报告 |
| **Phase 3** | 校准技术开发 | 校准流程 + 验证 |
| **Phase 4** | 系统集成试验 | 湖试→浅海→深海 |

## Phase 1 任务清单

### P0（立即执行）

- [ ] 声线跟踪完善：`ray_trace.m` 迭代保护 + 自适应分层（[[yangbaoguo-2013-usbl-calibration]] 方法）
- [ ] 声速剖面处理：SVP 读取/平滑/拓延/有效声速
- [ ] iUSBL 坐标变换：逆向自定位完整变换链推导与实现

### P1（本周）

- [ ] 改进 GCC 时延估计（[[huangjian-2019-lbl-usbl]] PHAT+功率谱加权）
- [ ] 基线分解 DOA 算法（[[dingjie-2020-compact-usbl]] 适配五元圆阵）
- [ ] `run_full_simulation.m` 集成声线跟踪和坐标变换

### P2（两周内）

- [ ] 安装校准算法框架：两步法 + M 估计（[[yangbaoguo-2013-usbl-calibration]]）
- [ ] Monte Carlo 评估框架：多场景自动化仿真
- [ ] 误差预算验证：各误差源独立注入

### P3（一个月内）

- [ ] 多径仿真模块：海面/海底反射
- [ ] 深度辅助定位：压力深度约束
- [ ] 动态仿真：运动平台连续定位

## Phase 2 仿真场景

| 场景 | 距离 | 深度 | SNR | 目的 |
|------|------|------|-----|------|
| 近距标称 | 1 km | 100 m | 20 dB | 基线验证 |
| 中距典型 | 5 km | 500 m | 10 dB | 典型性能 |
| 远距极限 | 10 km | 1000 m | 0 dB | 边界性能 |
| 浅海多径 | 3 km | 50 m | 15 dB | 多径鲁棒 |
| 大角度 | 5 km | 200 m | 10 dB | 宽角覆盖 |
| 动态 | 5 km | 300 m | 10 dB | 运动平台 |

## 误差预算

| 误差源 | 预算 | 验证方法 |
|--------|------|---------|
| DOA 估计 | ≤ 0.3° RMS | ML CRB + Monte Carlo |
| 测距 | ≤ 0.1%R | 匹配滤波仿真 |
| 声速修正 | ≤ 0.5 m/s | 声线跟踪仿真 |
| 姿态测量 | ≤ 0.1°/0.5° | 灵敏度分析 |
| 阵型误差 | ≤ 0.5 mm | 校准仿真 |
| 安装偏差 | ≤ 0.1° | 校准仿真 |
| **综合 RSS** | **< 1%R @10km** | 全链路仿真 |

## 风险与缓解

| 风险 | 缓解 |
|------|------|
| 10kHz 海洋噪声 SNR 不足 | LFM 脉压增益 26dB + 改进 GCC |
| d/λ=1.33 相位模糊残余 | ML 全局搜索 + CBF 交叉验证 |
| 远距声线弯曲 | 自适应分层声线跟踪 + CTD 实测 |
| ML 模型失配 | 基线分解法备选 + 多算法融合 |
| 校准精度不足 | 两步法 + M 估计 + 多航迹验证 |

## 里程碑

| 节点 | 交付 | 验收标准 |
|------|------|---------|
| M1 | MATLAB 全链路仿真 | 理想条件 < 0.5%R |
| M2 | Monte Carlo 报告 | 典型条件 < 1%R (N=1000) |
| M3 | 校准仿真报告 | 安装角精度 < 0.1° |
| M4 | 湖试报告 | 2km 内 < 1%R |
| M5 | 浅海报告 | 5km 内 < 1%R |
| M6 | 验收报告 | 10km 内 < 1%R |
