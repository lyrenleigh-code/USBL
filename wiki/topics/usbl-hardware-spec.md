# USBL 硬件自研专题

> 与算法线并行推进的硬件研制专题页。对应 [[lr-usbl-development-plan]] 中「硬件研制并行线」章节的详细展开。
>
> 技术基线参考：[[yumin-2006-lr-usbl]]（PZT4 复合棒 + 阵列集成 + 耐压 4000 m）、[[dingjie-2020-compact-usbl]]（阵型校准）、[[zhengcuie-usbl-docking]]（应答器）。

## HW-1 接收换能器

### 指标

| 参数 | 目标 | 说明 |
|------|------|------|
| 中心频率 | 10 kHz | 对应 λ=15 cm |
| −3 dB 带宽 | ≥ 4 kHz（40% 相对带宽） | LFM 脉压增益需求 |
| 接收灵敏度 | ≥ −190 dB re 1V/μPa | 远距低 SNR |
| 最大耐压 | 40 MPa（4000 m） | 系统深度包络 |
| 水平指向性 | 全向（单元） | 阵列合成方向性 |
| 垂直指向性 | ±60° 波束内平坦 | 大俯仰角覆盖 |
| 阻抗匹配 | 前放输入阻抗匹配 | 低噪 |
| 尺寸 | 单元 ≤ φ40 × 80 mm | 阵列封装空间 |

### 结构方案

- **核心**：PZT4 复合棒换能器（参考 [[yumin-2006-lr-usbl]]）
- **匹配**：前匹配层（声阻抗 ≈ √(Z_piezo × Z_water)），必要时分段匹配
- **尾质量**：钨合金块，提升低频响应
- **封装**：聚氨酯灌封 + 前端陶瓷/玻璃钢窗口
- **备选**：若复合棒带宽不够，考虑 1-3 压电复合材料（天然宽带）

### 风险与缓解

- **Q 值与灵敏度矛盾**：低 Q（宽带）牺牲灵敏度 → 仿真优化匹配层厚度/密度；必要时接收灵敏度指标放宽 3 dB 由算法侧补偿
- **温度漂移**：陶瓷谐振频率随温度漂移 → 冻结参数时留 ±5% 裕量

## HW-2 五元 UCA 阵列封装

### 指标

| 参数 | 目标 | 说明 |
|------|------|------|
| 阵型 | 5 元均匀圆阵 + 1 中心（可选） | d=20cm |
| 阵元间距误差 | ≤ ±0.5 mm | 对应 DOA 偏差 < 0.1° |
| 幅度一致性 | ≤ 1 dB | 筛选 + 校正表 |
| 相位一致性 | ≤ 5° @10kHz | 筛选 + 校正表 |
| 阵面平面度 | ≤ 1 mm | 平面阵假设前提 |
| 整体耐压 | 40 MPa | 同 HW-1 |
| 耦合隔离 | 阵元间 ≥ 30 dB | 避免互耦影响 DOA |

### 结构方案

- **基座**：钛合金/316L 不锈钢，精密加工保证阵型公差
- **减耦**：阵元间吸声层 + 去耦槽
- **屏蔽**：电磁屏蔽网 + 同轴电缆单独屏蔽
- **标定接口**：预留水池标定用校准接入口

### 制造与标定流程

1. 换能器批量筛选（N=10，选 5 只匹配最优）
2. 精密装配至基座
3. 整体耐压测试（常温分级 → 极限打压）
4. 水池电声标定：逐元发射响应 + 阵列接收一致性
5. 生成幅相校正表（存入干端设备）
6. 年度复校 SOP（[[yangbaoguo-2013-usbl-calibration]] 参考）

## HW-3 应答器

### 指标

| 参数 | 目标 | 说明 |
|------|------|------|
| 发射频率 | 10 kHz ±2 kHz | 匹配接收阵 |
| 发射源级 SL | ≥ 185 dB re 1μPa@1m | 10 km 链路预算 |
| 信号体制 | LFM chirp, BW=4 kHz, T=100 ms | 与接收匹配 |
| 应答模式 | 外触发/定时自发 | 双模 |
| 电池续航 | ≥ 30 天（1 Hz 应答） | 潜标系留需求 |
| 耐压 | 40 MPa | 同系统 |
| 授时 | TCXO + 定期 GPS 对时（部署前） | 时基精度 |
| 唯一标识 | CDMA 码或 chirp 斜率编码 | 多潜标区分 |

### 组成

- **发射换能器**：复合棒 / 环型（不同于接收单元，侧重大功率）
- **功放**：E 类开关功放（高效）
- **数字板**：MCU + DDS/DAC + 授时模块
- **电池仓**：锂亚一次电池或可充锂离子
- **结构**：独立耐压壳 + 系留接口

### 风险

- 185 dB 源级在 10 kHz 功率密度高，换能器空化阈值需校核
- 长期漂移：TCXO 月漂 ±1 ppm → 授时误差，部署前必须 GPS 对时
- 多目标扩展：CDMA/正交 chirp 预留（[[zhengcuie-usbl-docking]] 思路）

## HW-4 多通道采集电子

### 指标

| 参数 | 目标 | 说明 |
|------|------|------|
| 通道数 | 5（+可扩展） | 对应阵元 |
| 采样率 | ≥ 100 kSPS/ch（过采样 10×） | DOA 相位精度 |
| ADC 位数 | ≥ 16 bit | 动态范围 |
| 通道间时延 | < 10 ns | 同步采样硬指标 |
| 前端噪声 | ≤ 1 nV/√Hz @10 kHz | 决定本底 |
| 可编程增益 | 0–60 dB | 远近兼顾 |
| 抗混叠滤波 | 10–15 kHz 带通 | 数字域二次滤波前提 |
| 时钟源 | FPGA 统一分发 + OCXO | 通道同步基础 |
| 授时 | GPS/PPS 输入 | 与应答器对齐 |

### 架构

```
[阵元×5] → [前放 + 带通] → [PGA] → [ADC×5] → [FPGA 聚合]
                                         ↓
                                    [以太网/USB → 干端]
                      [OCXO/PPS] ──→ FPGA 时钟分发
```

### 关键实现

- **前放**：低噪运放（AD797 级）+ 差分输入
- **ADC**：AD7768 或 ADS127L01 类 Σ-Δ ADC，天然同步
- **FPGA**：Xilinx Artix-7 / Zynq，触发+时钟分发+数据聚合
- **接口**：千兆以太网 → 干端，时间戳随数据上行

## HW-5 结构与水密

### 清单

| 部件 | 材料/方案 | 关键指标 |
|------|---------|---------|
| 阵列耐压壳 | 钛合金 TC4 / 316L | 40 MPa，寿命 10 年 |
| 前端声窗 | 聚氨酯 / 玻璃钢 | 声透明 + 耐压 |
| 水密电缆 | 聚氨酯护套 + Kevlar 加强 | 4000 m，拉力 ≥ 2 kN |
| 水密连接器 | Subconn / 国产替代 | 40 MPa，插拔寿命 ≥ 500 次 |
| 甲板单元 | 标准机箱（IP65） | 干端处理+显控 |
| 安装基座 | 船底法兰 / 可拆卸 | 对心公差 ≤ 1 mm |
| 灌封材料 | 聚氨酯 / 环氧 | 声阻抗匹配 + 耐压 |

### 测试

- 分级打压：100 m → 1000 m → 4000 m → 5000 m（1.25× 余量）
- 温循：−5 °C ~ +40 °C × 20 次
- 盐雾：96 h
- 长期浸泡：72 h @常压（检漏）

## HW-6 系统联调与干端软件

### 内容

- 整机电声通道标定（幅相校正表写入）
- 硬算闭环：实时采集 → 算法 → 定位输出
- 时间同步验证（应答器 ↔ 采集 < 1 ms）
- 动态测试（模拟航迹）
- 显控界面：斜距/DOA/定位/SNR 实时显示

### 交付物

- 标定证书（每套设备）
- 联调测试报告
- 用户操作手册
- 维护保养 SOP

## 风险清单

| 风险 | 严重度 | 缓解 |
|------|-------|------|
| 10 kHz 宽带换能器做不出目标 BW | 🔴 | 方案备选（1-3 复合）+ 算法侧容差 |
| 阵元幅相一致性不达标 | 🔴 | 筛选 + 校正表 + 基线重构算法兜底 |
| 采集电子通道时延抖动 | 🟠 | Σ-Δ ADC 天然同步 + FPGA 校准 |
| 耐压壳加工周期长 | 🟠 | Phase 2 即下单，并行推进 |
| 水密电缆/连接器进口受限 | 🟠 | 早期接触国产替代厂商 |
| 应答器电池续航不足 | 🟡 | 降低应答频率或增大电池仓 |
| 外协质量不稳定 | 🟡 | 双供应商策略 + 入厂检验 |
