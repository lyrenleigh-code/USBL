# 超短基线系统定位精度改进方法研究

> **引用**: 蘧振超 (2024). 期刊论文. 浙江省测绘科学技术研究院.

## 核心贡献

提出五元十字阵多子阵分解融合定位方法（USBLall），通过多子阵独立定位+异常点剔除+LS迭代实现精度提升。

## 方法/算法

- **子阵分解**：五元十字阵拆分为8个三元子阵 + 1个四元子阵 + 1个五元子阵
- **融合流程**：各子阵独立定位 → 欧氏距离确定聚类中心 → eta参数异常点剔除 → LS迭代求解 → USBLall输出
- **异常点剔除**：基于eta参数的自适应阈值

## 关键参数/结果

| 参数 | 值 |
|------|-----|
| 阵元间距 | D = 0.10 m |
| 中心频率 | fc = 13.5 kHz |
| 带宽 | B = 3 kHz |
| 测试距离 | R = 1 km |

- USBLall在所有SNR条件下定位精度均为最高
- 融合方法对单子阵异常值具有鲁棒性
