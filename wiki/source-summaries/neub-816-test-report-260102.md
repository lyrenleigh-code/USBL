2. M2 相位残差 spec 放宽到 ≤4° 还是软件侧做通道校正（用本报告数据反补）？

### 指向性曲线

- **水平指向性（12 kHz）**：近似全向（水平面内各方位响应差 < 3 dB）
- **垂直指向性（12 kHz）**：**明显的阵法线方向指向性**
  - 主瓣 0°（阵法线方向）附近响应最强
  - 明显的非对称性（180° 方向有凹陷 / 零点）
  - 典型 cardioid 形态

**对阵型性质的启示**：
- 水平近似全向 → 支持"5 阵元绕中轴近似旋转对称"的立体阵假设（即 4 外围均布 + 1 中央）
- 垂直非对称 → **强烈暗示阵列有 top/bottom 不对称**，与工程图"中央 80 / 外围 110"的双高度布局一致 → **立体阵 confirmed**

## 核心观点

1. **工作频率实际是 12 kHz**（不是 USBL 项目一直假设的 10 kHz）。必须全项目级同步。

2. **阵型立体阵 confirmed**：垂直指向性非对称直接证明 5 阵元不在同一平面。

3. **M2 通道一致性 spec 需放宽**：实测两样机相位残差最大 ±3.6°，远超当前 M2 spec ≤1°。选择：
   - 硬件路径：要求供应商提高一致性（成本/周期压力）
   - 软件路径：**用本报告数据在 M2 校正表中反补**（推荐）——所有 5 通道的相位-频率 + 幅度-频率函数已有，接收链前置校正可将残差降至 < 0.5°

4. **H1 spec 需大幅改写**：
   - spec 目标 -190 dB 灵敏度 → 实测 **-200 dB**（优于 spec 10 dB）
   - spec 尺寸限制 φ40 × 80 mm（单元） → 实际为**阵列整体**（5 通道一同测试），非独立单元
   - spec 目标"耐压 40 MPa" → 测试报告未涉及耐压（驻波管 / 水池测试深度仅 0.2-0.6 m）——耐压测试缺失，需单独安排

5. **此报告不含阵列几何坐标**：依然需要供应商另行提供 STEP/IGES 模型或坐标表才能确定 5 阵元的 (x, y, z) 精确值。
