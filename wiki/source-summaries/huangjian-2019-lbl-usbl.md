# 长基线/超短基线组合水声定位跟踪关键技术研究

> **引用**: 黄健 (2019). 博士论文. 西北工业大学.

## 核心贡献

系统研究LBL/USBL组合定位的六项关键技术，涵盖时延估计、声速估计、定位解算、异常点处理和跟踪滤波全链路。

## 方法/算法

1. **改进GCC加权**：PHAT + 功率谱联合加权，提升低SNR下互相关峰锐度
2. **有效声速估计**：基于BELLHOP声传播模型，比平均声速法更准确
3. **PSO未知声速定位**：粒子群优化在声速未知时联合估计位置和声速
4. **区域划分自适应PSO**：根据搜索区域划分策略加速收敛
5. **Grubbs异常点剔除**：统计检验 + 距离加权融合LBL/USBL定位结果
6. **跟踪滤波**：UKF + R-T-S区间平滑，前向滤波+后向平滑

## 关键参数/结果

- 改进GCC加权在SNR < 0dB时时延估计成功率显著提升
- BELLHOP有效声速在远距（>5km）定位误差减少30%+
- UKF+RTS平滑后轨迹精度优于纯滤波

## 与项目的关联

| 关联模块 | 说明 |
|---------|------|
| 时延估计 | 改进GCC加权对远距低SNR场景直接可用 |
| 声速处理 | BELLHOP有效声速估计对10km远距定位价值大 |
| 跟踪滤波 | UKF+RTS平滑可用于连续定位输出 |
